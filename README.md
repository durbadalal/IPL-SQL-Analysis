# IPL 2018 vs IPL 2019 SQL Analysis

## Project Overview

This project analyzes IPL 2018 and IPL 2019 datasets using SQL to uncover trends in:

* Team performance
* Batting aggression
* Player impact
* Bowling effectiveness
* Strategic evolution across seasons

The objective was to transform raw cricket datasets into insight-driven analytical storytelling.

---

# Datasets Used

* IPL 2018 Batsmen
* IPL 2018 Bowlers
* IPL 2019 Batsmen
* IPL 2019 Bowlers

---

# SQL Concepts Demonstrated

* SELECT
* GROUP BY
* ORDER BY
* Aggregations
* CASE WHEN
* Ranking Functions
* Comparative Analysis

---

# Key Insights

* IPL 2019 showed significantly higher batting aggression
* Strike rates increased across top players
* Consistent teams outperformed star-dependent teams
* Bowlers faced greater scoring pressure in 2019
